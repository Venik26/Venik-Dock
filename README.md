# Venik26 Dock for Rainmeter

**English** | [繁體中文](README.zh-TW.md)  

A Pure Native Rainmeter Dock with Zero .ini Editing Required.

---

## Features

* **Smart Auto Icon Extractor (PowerShell & .NET API)**  
  Click any `.exe`, `.lnk`, or PWA to extract high-res (256x256) icons automatically. No manual PNG cropping needed.  

* **Scroll-to-Adjust Transparency & Custom Backgrounds**  
  Scroll mouse wheel to tweak background transparency in real-time, or easily apply your custom images.  

* **Adaptive Dynamic Grid Layouts (1 to 24 Columns)**  
  Right-click to dynamically switch layouts via pure math. Backgrounds and icons scale automatically.  

* **100% Pure Native Logic (Zero .ini Editing)**  
  Built entirely using native Rainmeter measures and formulas. Zero code-tampering required for users.

## Requirements

* **Rainmeter** (v4.5+) on **Windows 10 / 11**

---

## Installation

1. Go to the **[Releases](https://github.com/Venik26/Venik-Dock/releases)** page.  

2. Download the latest `Venik Dock.rmskin` file.  

3. Double-click the file to install automatically via Rainmeter.
