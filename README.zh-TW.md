# Venik Dock for Rainmeter

[English](README.md) | **繁體中文**

一個簡單自訂、免翻設定檔、流暢的純原生 `.ini` Rainmeter 應用程式Dock。
 
---

## 核心特色

* **點擊即可自動使用PowerShell提取高解析度圖標，免去手動找去背圖的痛苦。**  
  
* **支援滾輪即時微調背景透明度，並可自由更換背景圖片。**   

* **純公式計算，右鍵一鍵切換多欄位排版，外觀自適應縮放。**  

* **完全採用 Rainmeter 原生公式，使用者不需要翻設定檔設定。**

## 安裝說明

2. 下載最新的 `Venik Dock.rmskin` 安裝包
